# 🎓 پروژه پیش‌بینی عملکرد دانش‌آموزان

## 📋 معرفی پروژه

این پروژه در چارچوب درس **داده‌کاوی** انجام شده و هدف آن پیش‌بینی عملکرد دانش‌آموزان (قبول یا رد) بر اساس ویژگی‌هایی مانند زمان مطالعه، حمایت خانواده، فعالیت‌های فوق‌برنامه و سایر عوامل اجتماعی است.

## 🎯 اهداف پروژه

- **طبقه‌بندی (Classification)**: پیش‌بینی سطح عملکرد دانش‌آموزان
- **خوشه‌بندی (Clustering)**: گروه‌بندی دانش‌آموزان به دسته‌های مشابه  
- **قوانین همبستگی (Association Rules)**: شناسایی الگوهای پنهان بین ویژگی‌ها

## 📊 ویژگی‌های دیتاست

### ویژگی‌های عددی:
- `Hours_Studied`: ساعات مطالعه در هفته
- `Previous_Scores`: نمرات قبلی دانش‌آموز
- `Sleep_Hours`: ساعات خواب
- `Sample_Question_Papers_Practiced`: تعداد آزمون‌های تمرینی

### ویژگی‌های دسته‌ای:
- `Gender`: جنسیت
- `Parental_Involvement`: میزان درگیری والدین
- `Access_to_Resources`: دسترسی به منابع آموزشی
- `Motivation_Level`: سطح انگیزه
- `Internet_Access`: دسترسی به اینترنت
- `Family_Income`: درآمد خانواده
- `Teacher_Quality`: کیفیت تدریس

### متغیر هدف:
- `Performance_Category`: سطح عملکرد (Poor/Average/Excellent)

## 🔬 مراحل پروژه

### 1️⃣ پیش‌پردازش داده‌ها
- بررسی مقادیر گمشده
- کدگذاری متغیرهای دسته‌ای (One-Hot Encoding)
- نرمال‌سازی ویژگی‌های عددی
- تقسیم داده‌ها به مجموعه آموزشی و تست

### 2️⃣ مدل‌های طبقه‌بندی
- **Decision Tree**: درخت تصمیم‌گیری
- **Random Forest**: جنگل تصادفی
- **SVM**: ماشین بردار پشتیبان
- **Logistic Regression**: رگرسیون لجستیک

#### معیارهای ارزیابی:
- Accuracy (دقت)
- Precision (دقت مثبت)
- Recall (احساس)
- F1-Score
- Cross-Validation

### 3️⃣ خوشه‌بندی
- **K-Means**: با تعیین تعداد بهینه خوشه‌ها
- **DBSCAN**: برای شناسایی نقاط پرت
- ارزیابی با Silhouette Score

### 4️⃣ قوانین همبستگی
- الگوریتم **Apriori** برای استخراج الگوهای پرتکرار
- محاسبه معیارهای Support، Confidence و Lift
- شناسایی روابط بین ویژگی‌ها و عملکرد

## 📊 نتایج کلیدی

### بهترین مدل طبقه‌بندی
- مدل: Random Forest
- دقت: ~85%
- F1-Score: ~0.84

### خوشه‌بندی
- تعداد خوشه بهینه: 3-4 خوشه
- Silhouette Score: ~0.6

### عوامل مؤثر (بر اساس همبستگی)
1. **نمرات قبلی** (بالاترین همبستگی)
2. **ساعات مطالعه**
3. **درگیری والدین**
4. **دسترسی به منابع**

## 💡 یافته‌های مهم

### 🌟 دانش‌آموزان برتر:
- میانگین ساعت مطالعه: 12+ ساعت
- نمرات قبلی: 85+ 
- درگیری بالای والدین: 70%+
- فعالیت فوق‌برنامه: 80%+

### 📈 پیشنهادات بهبود:
1. **افزایش ساعات مطالعه**: مؤثرترین عامل
2. **بهبود کیفیت خواب**: 6-8 ساعت در شب
3. **تقویت حمایت خانوادگی**: درگیری والدین
4. **دسترسی بهتر به منابع**: کتاب، اینترنت، امکانات

## 🔍 تحلیل خوشه‌ها

### خوشه 1: دانش‌آموزان برتر
- عملکرد غالب: Excellent
- ساعت مطالعه بالا
- حمایت قوی والدین

### خوشه 2: دانش‌آموزان متوسط  
- عملکرد غالب: Average
- ساعت مطالعه متوسط
- حمایت متوسط والدین

### خوشه 3: دانش‌آموزان ضعیف
- عملکرد غالب: Poor
- ساعت مطالعه کم
- حمایت کم والدین

## 📈 قوانین همبستگی جالب

```
ساعت مطالعه زیاد + حمایت والدین بالا → عملکرد عالی (Confidence: 75%)
نمره قبلی عالی + فعالیت فوق‌برنامه → عملکرد عالی (Confidence: 80%)
خواب کم + استرس بالا → عملکرد ضعیف (Confidence: 65%)
```
